## 🛠️ Technologies Used
*   **HTML5:** Semantic structural layout.

## 🌟 Key Features Demonstrated
*   **Responsive Web Design (RWD):** Fluid layout that adapts seamlessly across mobile, tablet, and desktop viewports.
*   **Consistent Theme Colors:** Uses a deep professional blue palette (inspired by academic trust) paired with accessible contrast elements to avoid overwhelming brightness.
*   **Real-World Skills Integration:** Showcases a strong blend of academic tech familiarity, operational business administration, and vital soft skills (customer relations, team leadership, and conflict resolution).
